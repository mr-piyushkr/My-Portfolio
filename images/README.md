# Images Folder Structure

## How to Add Your Images

### 1. Profile Images (`/images/profile/`)

- **profile.jpg** - Your main profile photo (recommended: 420x420px or larger, square ratio)
- Use JPG or PNG format

### 2. Project Images (`/images/projects/`)

- Add your project screenshots/images here
- Name them descriptively: `project-name-1.jpg`, `project-name-2.jpg`, etc.
- Recommended size: 1200x800px or similar aspect ratio

### 3. Logo Images (`/images/logos/`)

- **project-logo.png** - Logo for your projects (recommended: 128x128px, transparent background)
- Add client logos or brand logos here

## Supported Formats

- JPG/JPEG (best for photos)
- PNG (best for logos with transparency)
- WebP (modern format, smaller file size)

## Tips

- Keep file sizes under 500KB for faster loading
- Use descriptive filenames (no spaces, use hyphens)
- Optimize images before uploading (use tools like TinyPNG)
